# Metal SDK

A modern Node.js SDK for Metal.

## Installation

```bash
npm install metal-sdk
```

## Usage

```typescript
import { MetalSDK } from 'metal-sdk';

const client = new MetalSDK({
  apiKey: 'your-api-key'
});

// Use the SDK
```

## Development

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the project:
   ```bash
   npm run build
   ```
4. Run tests:
   ```bash
   npm test
   ```
